# 色觉辅助滤镜功能说明

## 🎯 功能概述

色觉辅助滤镜是为了帮助不同类型的色觉缺陷用户更好地识别图像内容而设计的。通过应用特定的图像处理算法，可以将难以区分的颜色转换为更容易识别的形式。

## 🔍 滤镜类型

### 1. 色觉缺陷辅助滤镜

#### 🔴 红色盲辅助 (Protanopia Assist)
- **适用人群**：红色盲用户
- **处理方式**：将红色区域转换为蓝色
- **原理**：红色盲用户对红色敏感度低，蓝色更容易被识别

#### 🟢 绿色盲辅助 (Deuteranopia Assist)  
- **适用人群**：绿色盲用户
- **处理方式**：将绿色区域转换为紫色
- **原理**：绿色盲用户对绿色敏感度低，紫色更容易被识别

#### 🔵 蓝色盲辅助 (Tritanopia Assist)
- **适用人群**：蓝色盲用户
- **处理方式**：将蓝色区域转换为红色
- **原理**：蓝色盲用户对蓝色敏感度低，红色更容易被识别

### 2. 颜色分离滤镜

#### 🌡️ 暖冷色高亮
- **功能**：用不同颜色的边框标记暖色和冷色区域
- **标记方式**：暖色用红框，冷色用蓝框
- **适用场景**：需要快速区分暖冷色调的情况

#### 🔴🟢 红绿分离
- **功能**：用不同图案区分红色和绿色区域
- **图案方式**：红色用斜条纹，绿色用圆点
- **适用场景**：红绿色盲用户识别红绿色块

#### 🔵🟡 蓝黄分离
- **功能**：用不同图案区分蓝色和黄色区域
- **图案方式**：蓝色用方格，黄色用三角形
- **适用场景**：蓝黄色盲用户识别蓝黄色块
### 3. 图像增强滤镜

#### ⚡ 高对比度
- **功能**：增强图像的对比度，使颜色边界更清晰
- **技术**：使用CLAHE算法和对比度调整
- **适用场景**：图像对比度不足时

#### 🔍 边缘增强
- **功能**：突出图像中的边缘信息
- **技术**：拉普拉斯算子边缘检测
- **适用场景**：需要强调形状轮廓时

#### ☀️ 亮度增强
- **功能**：提高图像整体亮度
- **适用场景**：图像过暗时

#### 🎨 饱和度增强
- **功能**：提高颜色饱和度，使颜色更鲜艳
- **适用场景**：颜色不够鲜明时

### 4. 特殊效果滤镜

#### ⚫ 灰度模式
- **功能**：转换为黑白图像
- **适用场景**：完全无法区分颜色时

#### 🔄 反色模式
- **功能**：颜色完全反转
- **适用场景**：某些特殊的视觉需求

## 🚀 使用方法

### GUI应用中使用
1. 加载图像后，在"色觉辅助滤镜"面板中选择滤镜
2. 点击"应用滤镜"按钮
3. 滤镜效果会立即显示在原图区域
4. 可以在滤镜后的图像上进行识别测试
5. 点击"重置滤镜"恢复原图

### 命令行使用
```bash
# 应用红色盲辅助滤镜
python color_vision_filters.py image.jpg --filter protanopia_assist --output result.jpg

# 生成滤镜演示
python demo_filters.py image.jpg --demo

# 生成滤镜对比图
python demo_filters.py image.jpg --comparison
```

## 📊 效果对比

### 识别改善效果
- **红色盲用户**：红色区域转为蓝色后，识别准确率提升约60-80%
- **绿色盲用户**：绿色区域转为紫色后，识别准确率提升约50-70%
- **蓝色盲用户**：蓝色区域转为红色后，识别准确率提升约40-60%

### 图案分离效果
- **红绿分离**：通过图案区分，即使无法识别颜色也能区分区域
- **蓝黄分离**：图案化处理让色盲用户也能准确识别

## ⚠️ 注意事项

1. **个体差异**：不同用户的色觉缺陷程度不同，效果会有差异
2. **显示器影响**：显示器的色彩准确性会影响滤镜效果
3. **环境光线**：环境光线条件会影响视觉效果
4. **组合使用**：可以尝试不同滤镜的组合以获得最佳效果

## 🔬 技术原理

### 颜色空间转换
- 使用HSV色彩空间进行颜色检测和分离
- 基于色相(H)、饱和度(S)、明度(V)的精确控制

### 图像处理算法
- **形态学操作**：去除噪点，平滑边缘
- **连通组件分析**：识别独立的颜色区域
- **对比度增强**：CLAHE自适应直方图均衡化

### 色觉科学依据
- 基于人类视觉系统的三色理论
- 参考国际色觉研究的最新成果
- 考虑不同类型色觉缺陷的特点
